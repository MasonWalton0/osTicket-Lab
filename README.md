# osTicket Ticketing System Lab

## Overview
This project demonstrates the deployment and configuration of an open-source ticketing system (osTicket) in a Linux-based virtual environment. The lab simulates a real-world IT help desk workflow, from ticket creation to resolution, using industry-standard practices.

## Lab Environment
- Ubuntu Server 22.04 LTS (Ticketing Server)
- Apache Web Server
- PHP
- MySQL Database
- osTicket (Open Source Ticketing System)
- Oracle VirtualBox observed for virtualization

## System Configuration
- Installed and configured Apache, PHP, and MySQL
- Deployed osTicket via web-based installer
- Secured database and application configuration
- Verified system accessibility from host browser

## Ticketing Features Implemented
- User and agent account creation
- Department and group configuration
- Role-based access control
- Service Level Agreements (SLAs)
- Ticket intake, assignment, escalation, and resolution

## Ticket Workflow Demonstrated
- End-user ticket submission
- Agent ticket assignment
- SLA-based prioritization
- Ticket updates and communication
- Ticket resolution and closure

## Skills Demonstrated
- Linux server administration
- Web application deployment
- Help desk ticketing systems
- Role and permission management
- SLA implementation
- IT documentation and workflow management

## Documentation
Detailed setup notes and screenshots are included to demonstrate each phase of the installation and ticket lifecycle.
