# osTicket Ticketing System Lab

## Description
Deployed and configured a full-featured open-source IT ticketing system using osTicket on Ubuntu Server.

## Skills Demonstrated
- Linux server administration
- Web server deployment (Apache, PHP, MySQL)
- Ticket lifecycle management
- SLA configuration
- Role-based access control

## Features Implemented
- Multiple departments and teams
- SLA-based ticket prioritization
- Agent escalation workflows
- End-user ticket submission
